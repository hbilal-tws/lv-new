
## Installation

Install Composer dependencies

```bash
  composer update/install
```
Copy envirmental file

```bash
  cp .env.example .env
```

Generate key

```bash
  php artisan key:gen
```

Migrate migrations

```bash
  php artisan migrate
```
Install Javascript dependencies

```bash
  npm install
```
Compile Javascript

```bash
  npm run dev
```
Watch for changes

```bash
  npm run watch
```
